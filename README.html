This is a practical direction to learn CS and AI for anyone who is interested in.

Index

  1. Learn an introduction to CS course
  2. Learn to code in (at least) one object-oriented programming language (C++, Java®, Python®)
  3. Learn other programming language
  4. Test your code
  5. Develop logical reasoning and knowledge of discrete math
  6. Develop a strong understanding of algorithms and data structures
  7. Machine Learning
  8. Computer Vision
  9. Natural Language Processing
  10. Reference

Introduction to CS

Focus on basic coding instructions. Online resources:

  • Introduction to Computer Science, Udacity
  • CS50x Harvard, Languages include C, Python, SQL, and JavaScript plus CSS and HTML

Object-oriented programming language

  • For beginning programmers:

    • Programming Methodology, Stanford CS106A, Java
    • Structure and Interpretation of Computer Programs, Berkeley University CS 61A, Python
    • Introduction to Programming in Java, MIT
    • Google’s Python Class
    • Google’s C++ Class

  • For more experienced programmers:

    • Data Structures, Berkeley University CS 61B, Java
    • Design of Computer Programs, Udacity, Python
    • Programming Abstractions, Stanford CS106B, C++
      Latest assignments: http://web.stanford.edu/class/cs106b/
    • Book: Data Structures & Algorithm Analysis in C++, 4th ed, by Mark A. Weiss

Test your code

Learn how to catch bugs, create tests, and break your software.

  • Software Testing, Udacity

Logical reasoning and knowledge of discrete math

  • Mathematics for Computer Science, MIT
  • Introduction to Mathematical Thinking, Stanford, Coursera
  • Probabilistic Graphical Models, Stanford, Coursera
  • Game Theory, Stanford and University of British Columbia, Coursera

Algorithms and data structures

Learn about fundamental data types (stack, queues, and bags), sorting algorithms (quicksort, mergesort, heapsort), data structures (binary search trees, red-black trees, hash tables), and Big O.

  • Introduction to algorithms, MIT, fall 2011
  • Algorithms, Princeton, Part 1 & Part2
  • Algorithms: Design and Analysis, Stanford, Part 1 & Part 2
  • Algorithms, 4th Edition by Robert Sedgewick and Kevin Wayne
  • Tool: AlgorithmVisualizer
